# VocoFlo Moment Coach — OpenAI Build Week 2026

A standalone responsive web beta created for OpenAI Build Week 2026.

## Working Build Week beta

The fixed beta loop is:

1. The user describes an upcoming speaking situation.
2. GPT-5.6 generates one bounded speaking mission.
3. The user reports what happened.
4. GPT-5.6 extracts useful evidence from the report.
5. GPT-5.6 gives one next step.
6. The browser saves a simple local history.

The application must clearly distinguish working beta functionality from planned VocoFlo capabilities.
